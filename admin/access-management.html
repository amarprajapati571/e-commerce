Access Level Descriptions
Full Access

Complete control including create, read, update, and delete operations. Can manage settings and configurations.

View & Edit

Can view all data and make edits to existing records. Cannot delete records or change system settings.

View Only

Read-only access to view data and reports. Cannot make any changes, additions, or deletions to any records.

No Access

Module is completely hidden and inaccessible. User cannot view, edit, or interact with this section at all.
